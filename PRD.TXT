1. OVERVIEW & OBJECTIVE
   Plotmaster is a professional, web-based scientific plotting application designed
   for researchers, clinicians, and data scientists. It provides a visual interface
   to import structured datasets, map variables, apply academic journal color schemes,
   and export publication-ready figures. The tool aims to replicate the clean and
   rigorous aesthetics of R's ggplot2 library and offer seamless integration with
   GraphPad Prism formats.

2. TARGET AUDIENCE & CRITICAL VALUES
   - Academic Researchers: Require high-resolution vector exports (SVG) and strict
     journal-compliant color palettes.
   - Clinical Scientists: Often work with GraphPad Prism (.prism, .pzfx) files and
     need to convert or visualize tables quickly.
   - Usability: Zero-install browser-based interface running entirely client-side,
     ensuring complete data privacy (no data is uploaded to remote servers).

3. FUNCTIONAL REQUIREMENTS

   3.1. DATA IMPORT & PARSING
        The system must support the following input formats:
        - CSV (.csv): Standard comma-separated tabular data.
        - Excel (.xlsx, .xls): Multiple worksheets, parsing numerical and text cells.
        - Markdown Tables (.md): Extracts data from standard markdown table syntax.
        - GraphPad Prism XML (.pzfx): Parses data tables, column headers, and subcolumns.
        - GraphPad Prism ZIP (.prism): Extracts CSV files from internal data directories.
        - Dynamic Worksheet Selector: When an uploaded file contains multiple sheets
          (Excel, Prism), a dropdown must allow the user to select which dataset to load.

   3.2. VARIABLE MAPPING (AESTHETIC MAPPINGS / AES)
        After data selection, users must be able to map active dataset columns to:
        - X-Axis: Categorical or numeric data.
        - Y-Axis: Numeric data (supports multi-column selection for multiple Y values).
        - Group/Color: Categorical column to segment data series and apply color palettes.

   3.3. PLOT TYPES (10 ggplot2-INSPIRED VISUALIZATIONS)
        1. Scatter Plot (散点图): Numeric X vs Numeric Y. Supports point size/opacity.
        2. Line Plot (折线图): Trend lines connecting data points. Supports line thickness.
        3. Bar Plot (柱状图): Category X vs Numeric Y. Supports grouped or stacked layout.
        4. Box Plot (箱线图): Shows min, Q1, median, Q3, max, and outliers.
        5. Violin Plot (小提琴图): Displays density distribution of numerical data.
        6. Histogram (直方图): Distribution of a single numerical variable.
        7. Bubble Plot (气泡图): Scatter plot where marker size maps to a third variable.
        8. Heatmap (热力图): Grid visualization for correlation or matrix data.
        9. Area Chart (面积图): Shaded area under a line plot.
        10. Pie/Donut Chart (饼图/环形图): Proportional distribution of categories.

   3.5. CUSTOM STYLE EDITOR
        Provides visual controls for:
        - Typography: Selection of Arial, Inter, Helvetica, Times New Roman, Roboto.
        - Text Sizes: Titles, Axis Labels, Ticks, Legends.
        - Plot Parameters: Marker Size, Line Width, Bar Gap/Padding, Box Width.
        - Gridlines: Show/Hide vertical/horizontal gridlines.
        - Theme Templates:
          - Classic (white background, thin black axes).
          - Minimal (no background, light gray gridlines).
          - Dark (dark slate background, glowing elements).
          - ggplot2 Grey (light gray background, white gridlines).

   3.6. PUBLICATION-READY EXPORT
        - JPG & PNG: Adjustable resolution (standard/high DPI).
        - SVG: Full XML-based vector export.
        - GraphPad Prism XML (.pzfx): Recreates the tabular structure with column
          relationships and stores it in Prism's open XML format.
        - GraphPad Prism ZIP (.prism): Creates a ZIP package containing the dataset
          exported as `data/tables/table1.csv` and a manifest `document.json`.

4. SYSTEM ARCHITECTURE & DATA FLOW
   - Client-Side Only: Runs fully in the user's browser via index.html. No backend server.
   - Excel Parsing: JS SheetJS (XLSX.js) library handles raw parsing.
   - Zip Extraction/Compression: JSZip library handles reading/writing `.prism` files.
   - Charting: Apache ECharts library powers the visualizations.

5. NON-FUNCTIONAL REQUIREMENTS
   - Performance: Immediate rendering for datasets up to 10,000 rows.
   - Security: Data privacy guaranteed as all processing occurs in local memory.
   - UI/UX: Rich aesthetics using a dark-theme glassmorphic interface, layout,
     hover effects, and clean, intuitive controls.
